# ChatGenius - AI-Powered Education Chatbot

ChatGenius is an intelligent assistant designed for Computer Science students and professionals. It specialized in CS topics, coding assistance, career guidance, and study help.

## Features
- **Intelligent CS Chat**: Real-time conversation with specialized CS knowledge.
- **Coding Help**: Code generation, debugging, and line-by-line explanations.
- **Career Roadmaps**: Personalized guidance for Web Dev, AI, Data Science, and more.
- **Study Materials**: Notes, summaries, and MCQs.

## Tech Stack
- **Frontend**: Vite + React
- **Backend**: Node.js + Express
- **AI**: Google Gemini 1.5 Flash

## Installation

### Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file with `GEMINI_API_KEY`.
4. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
